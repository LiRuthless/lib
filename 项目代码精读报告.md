## 一、项目整体架构

本项目是一个典型的**电磁循迹智能车**控制系统，采用**双闭环 PID 控制架构**：

- **外环（方向环）**：以 50ms 为周期，通过四路电感采集赛道电磁线信号，计算循迹偏差，经位置式 PID 输出方向修正量，差速分配给左右轮目标速度。
- **内环（速度环）**：以 10ms 为周期，通过编码器反馈实际速度，经位置式 PID（带死区前馈）输出 PWM 控制电机。

```
┌─────────────────────────────────────────────────────────────────┐
│                         系统架构概览                              │
├─────────────────────────────────────────────────────────────────┤
│  传感器层  │  四路电磁电感(ADC)  │  编码器  │  六轴IMU(预留)   │
├─────────────────────────────────────────────────────────────────┤
│  控制算法层│  循迹PID(50ms)     │  速度PID(10ms)                 │
├─────────────────────────────────────────────────────────────────┤
│  执行层   │  左右轮电机(PWM+DIR) │  负压电机(PWM)                │
├─────────────────────────────────────────────────────────────────┤
│  交互层   │  IPS114显示屏      │  ADC按键菜单  │  串口/无线调参  │

│  存储层   │  片内Flash(EEPROM) 参数掉电保存                       │

```

## 三、核心工作原理

### 3.1 双中断时基架构

系统配置了两个周期性中断（PIT，Periodic Interrupt Timer），构成控制系统的**双时钟节拍**：

| 中断源 | 周期 | 服务函数 | 职责 |
|--------|------|----------|------|
| **TIM3 PIT** | **50ms** | `pit_track()` | 循迹控制：采样电感 → 计算偏差 → 循迹PID → 差速分配目标速度 |
| **TIM4 PIT** | **10ms** | `pit_speed()` | 速度闭环：读编码器 → 速度PID → 输出电机PWM + **喂看门狗** |

**关键设计要点**：
- 速度环周期（10ms）远快于方向环（50ms），符合内环快、外环慢的经典控制理论。
- 看门狗喂狗放在 **10ms 中断**中，确保即使主循环因菜单操作阻塞，系统也不会复位。
- 看门狗配置：`WDT_CONTR = 0x21`，预分频 4，溢出时间约 **140ms**。

### 3.2 电磁循迹原理

#### 3.2.1 电感布局与ADC通道

```
电感物理布局（车头视角，从左到右）：

    电感1(adc0)      电感2(adc1)      电感3(adc2)      电感4(adc3)
    [外侧左]         [内侧左]         [内侧右]         [外侧右]
      │                │                │                │
   ADC_CH13         ADC_CH14          ADC_CH8          ADC_CH10
    P05              P06               P00              P02
```

- 四路电感水平排列，外侧电感（adc0/adc3）感受距离远、内侧电感（adc1/adc2）分辨率高。
- 采用 **12位 ADC** 分辨率，量程 0~4095。

#### 3.2.2 去极值平均滤波

`read_adc()` 对每路通道采样 **8 次**，去掉最大值和最小值，剩余 6 次取平均：

```c
adc_sum[i] -= max[i];            // 去掉最大值
adc_sum[i] -= min[i];            // 去掉最小值
adc_filted[i] = adc_sum[i] / 6;  // 6次平均
```

该滤波能有效消除电磁干扰引起的偶然尖峰噪声。

#### 3.2.3 归一化循迹偏差计算

`get_track_error()` 采用**差分放大**原理计算归一化偏差：

```c
left_diff  = adc_filted[0] - adc_filted[3];   // 外侧电感差
right_diff = adc_filted[1] - adc_filted[2];   // 内侧电感差
denominator = (adc_filted[0] + adc_filted[3]) + labs(adc_filted[1] - adc_filted[2]);

diff = 1000 * (left_diff + 5 * right_diff) / denominator;
```

| 符号 | 含义 |
|------|------|
| `left_diff` | 外侧电感差分，反映大幅偏离 |
| `right_diff` | 内侧电感差分，反映小幅偏离 |
| `5 * right_diff` | 内侧电感加权 5 倍，提高近中心分辨率 |
| `denominator` | 归一化分母（外侧和 + 内侧差绝对值），消除赛道电流强度影响 |
| 返回值 | 范围约 **[-1000, 1000]**，负值偏左、正值偏右 |

### 3.3 双闭环 PID 控制详解

#### 3.3.1 外环：循迹方向 PID（位置式）

```c
int16 PID_track(void)
{
    P_out  = KP_x  * error;                          // 线性比例
    P2_out = K2P_x * error * abs(error);             // 非线性二次项
    D_out  = KD_x  * (error - error_last);           // 微分项
    
    PID_out = P_out + P2_out + D_out;
    
    // 输出限幅 ±MAX_DIR_OUT (±200)
}
```

**控制特性**：
- **非线性 P² 项**：`K2P_x * error * |error|`，小偏差时响应柔和，大偏差时快速修正。
- **无积分项**：方向环通常不需要积分，避免在弯道上持续累计。
- **输出限幅**：±200，作为差速基准值。

#### 3.3.2 差速控制策略

```c
if(track_out >= 0) {    // 偏右，需左转
    target_speed_L = base_speed - 3 * track_out;   // 左轮大幅减速
    target_speed_R = base_speed + 1 * track_out;   // 右轮小幅加速
} else {                // 偏左，需右转
    target_speed_L = base_speed - 1 * track_out;   // 左轮小幅加速
    target_speed_R = base_speed + 3 * track_out;   // 右轮大幅减速
}
```

- **非对称差速**：转向时内侧轮减速倍数（3×）大于外侧轮加速倍数（1×），防止速度过高冲出赛道。
- `base_speed` 为基础目标速度，由调参页面设定。

#### 3.3.3 内环：左右轮速度 PID（位置式，含死区前馈）

当前实际使用的是 **位置式 PID**（`PID_L_pos / PID_R_pos`）：

```c
errorL = target_speed_L - real_speed_L;

P_outL = KP_v * errorL;
I_outL += KI_v * errorL;      // 积分累计

// 积分限幅：给死区前馈留出余量，防止暗饱和
if(I_outL >  MAX_SPD_OUT - MOTOR_DEAD_ZONE) ...

PID_outL = P_outL + I_outL + D_outL;

// 死区前馈：根据目标方向叠加偏置，target=0时不补偿
if(target_speed_L > 0) PID_outL += MOTOR_DEAD_ZONE;
if(target_speed_L < 0) PID_outL -= MOTOR_DEAD_ZONE;

// 输出限幅 ±MAX_SPD_OUT (±7000)
```

**关键设计**：
- **死区前馈（MOTOR_DEAD_ZONE = 1300）**：直流电机在 PWM 占空比过低时不转或抖动，通过死区补偿直接抬升到响应阈值，确保电机灵敏启动。
- **目标为0时清零偏置**：确保停车命令能真正执行，不会因死区补偿而自转。
- **方向切换清零积分**：目标速度变号时清零积分项，避免反向拖尾。

### 3.4 编码器速度采集与滤波

```c
void read_encoder(void)
{
    encoder_L = -encoder_get_count(ENCODER_DIR_L);   // 读取左轮编码器（取反修正方向）
    encoder_R =  encoder_get_count(ENCODER_DIR_R);   // 读取右轮编码器
    
    // 一阶低通滤波：y[n] = alpha * x[n] + (1-alpha) * y[n-1]
    real_speed_L = lowpass_update(&filt_encoder_L, encoder_L);
    real_speed_R = lowpass_update(&filt_encoder_R, encoder_R);
    
    // 累计行驶距离
    Distance = (distance_L + distance_R) / 2;
    
    encoder_clear_count(...);   // 清零计数，准备下一次采样
}
```

- **滤波系数 alpha = 0.65**：平衡响应速度与噪声抑制。
- 编码器每 10ms 采样一次，计数清零制，实际得到的是**脉冲数/10ms**的瞬时速度量。

### 3.5 电机驱动输出

```c
// 左轮
if(speed_outL >= 0) {
    gpio_set_level(MOTOR_DIR_L, GPIO_HIGH);   // 正转
    pwm_set_duty(MOTOR_PWM_L, speed_outL);    // 设置占空比
} else {
    gpio_set_level(MOTOR_DIR_L, GPIO_LOW);    // 反转
    pwm_set_duty(MOTOR_PWM_L, -speed_outL);   // 取绝对值
}
```

- **PWM 频率 17KHz**：避开人耳可听范围，降低电机啸叫。
- 左右轮 DIR 逻辑相反（左轮 HIGH=正转，右轮 LOW=正转），匹配电机物理安装方向。
- **负压电机**（MOTOR_PWM_M）：独立 PWM 控制，用于产生下压力增加抓地力。

### 4.2 启动/停止逻辑（key_start）

```c
void key_start(void)
{
    // 1. 检测 IO_P22 按键下降沿，翻转 Start 标志
    if(last_key && !cur_key) Start = !Start;
    
    // 2. 电池电压检测（ADC_CH11_P03）
    battery_filt = lowpass_update(&filt_battery, battery);
    
    // 3. 低压保护：低于阈值时蜂鸣器报警并强制停止
    while(battery < 1200 && battery > 300)
    {
        Start = 0;
        P10 = 1; delay(100ms); P10 = 0; delay(100ms);  // 蜂鸣器闪烁
    }
    
    // 4. 启动瞬间清零速度PID累计输出，防止启动冲击
    if(Start) { PID_outL = 0.0; PID_outR = 0.0; }
}
```

## 五、人机交互系统

### 5.1 IPS114 菜单系统

菜单采用**页面+光标**的二级结构：

| 页面 ID | 名称 | 内容 |
|---------|------|------|
| 0 | 主页 (PAGE_HOME) | ADC_ERR / SPD_DIS / GYRO / (ADJUST1/ADJUST2) |
| 21 | ADC实时 (PAGE_ADC_ERR) | 四路电感ADC值 |
| 22 | 速度距离 (PAGE_SPD_DIS) | 左右轮速度、行驶距离 |
| 23 | 陀螺仪 (PAGE_GYRO) | 三轴角速度 |
| 11 | 调参页1 (PAGE_ADJUST1) | 慢速模式参数（自动保存到地址0x210） |
| 12 | 调参页2 (PAGE_ADJUST2) | 快速模式参数（自动保存到地址0x230） |

### 5.2 ADC 按键输入

两路 ADC 分压键盘实现 10 个按键：

```
KEY_CHANNEL1 (ADC_CH4_P14):  UP / DOWN / OK / LEFT / RIGHT
KEY_CHANNEL2 (ADC_CH3_P13):  RST / ADJUST1 / ADJUST2 / FOUR / BACK
```

按键值通过分压电阻产生特定电压区间，经 ADC 读取后除以 100 识别。

### 5.3 调参操作

在调参页面（PAGE_ADJUST1 / PAGE_ADJUST2）中：
- **上下键**移动光标（KP_x / K2P_x / KD_x / base_speed / fan_duty）
- **左右键**微调参数（步进 ±0.001 或 ±100）
- 页面切换时**自动写入 EEPROM**

### 5.4 一键模式切换

- **ADJUST1 键**：一键加载"平稳模式"（写入并读取地址 0x210 参数）
- **ADJUST2 键**：一键加载"快速模式"（写入并读取地址 0x230 参数）

## 六、参数存储（EEPROM / IAP）

利用 AI8051U **片内 Flash 模拟 EEPROM**（IAP 功能），实现参数掉电保存：

| Flash 地址 | 存储内容 | 操作函数 |
|-----------|---------|---------|
| 0x0110~0x0112 | 页面(page) + 光标(arrow) | `read_menu / write_menu` |
| 0x0210~0x0218 | 慢速模式 PID + 速度参数 | `read_speed_low / write_speed_low` |
| 0x0230~0x0238 | 快速模式 PID + 速度参数 | `read_speed_high / write_speed_high` |

> 注：每次写 Flash 前需调用 `iap_init()`，擦写寿命约 10 万次。

## 七、预留扩展：六轴 IMU

`gyroscope.c` 已集成 **IMU660RB** 六轴传感器接口，但目前主控制逻辑中**未启用**。

现有功能：
- `read_gyro_angle()`：读取陀螺仪并积分得到角度
- `read_accel_velocity()`：读取加速度并积分得到速度
- `gyro_calibrate() / accel_calibrate()`：静态零偏校准（100次采样平均）

潜在扩展方向：
- 利用陀螺仪 Z 轴（yaw 角速度）辅助判断过弯速率
- 利用加速度计检测坡道，进行坡道速度补偿
- 融合电磁+惯性导航，在电磁线断点处短时盲跑

## 九、代码设计亮点

1. **双中断解耦**：方向环与速度环分离到不同时基中断，避免控制耦合，便于独立调参。
2. **喂狗独立于主循环**：防止菜单或串口阻塞导致看门狗复位，提高系统可靠性。
3. **死区前馈 + 零速清零**：解决直流电机小信号死区问题，同时保证停车可靠。
4. **非线性 P² 项**：`error * abs(error)` 在循迹 PID 中提供自适应增益，小弯柔和、大弯迅猛。
5. **非对称差速**：内轮减速 3×、外轮加速 1×，符合物理直觉，降低甩尾风险。
6. **去极值平均滤波**：8 采样去头去尾，硬件成本为零而滤波效果显著。
7. **EEPROM 掉电保存**：两套参数一键切换，方便赛道现场快速调整。
8. **电池低压保护**：蜂鸣器报警 + 强制停车，防止电池过放损坏。
